# TauTranslatorOmega PWA

This directory contains the Progressive Web App (PWA) for TauTranslatorOmega, built with React and Next.js.

## Prerequisites

- Node.js (v18.x or later recommended)
- npm or yarn

## Getting Started

1.  **Navigate to the PWA directory:**
    ```bash
    cd ~/TauTranslator/pwa
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    # yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    # yarn dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

-   `components/`: Reusable React components.
-   `pages/`: Application pages (routes).
-   `public/`: Static assets (icons, manifest).
-   `styles/`: CSS files (global and modules).
-   `next.config.js`: Next.js configuration.
-   `package.json`: Project dependencies and scripts.
